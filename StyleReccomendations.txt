🧩 Design System Recommendations for VLabs Experiments – Santhoshini Thota
Foundations
🎨 Colors
Recommended Values:
•	Primary Palette: Blue gradients (#3182ce, #4299e1) for headers and primary actions
•	Success: Green (#10b981, #48bb78)
•	Warning: Orange (#f59e0b, #ed8936)
•	Error: Red (#ef4444, #f56565)
•	Neutral: Grayscale (#2d3748, #64748b, #f8fafc)
•	Background Branding: Purple gradient (#667eea, #764ba2)
Design Notes:
✅ Consistent semantic use, strong accessibility contrast
🅰 Typography
Recommended Values:
•	Font: Inter (with system font fallback)
•	Hierarchy: H1 (2em) → H3 (1.1em) → Body (0.875rem)
•	Weights: 700 (headers), 600 (subheads), 500 (labels), 400 (body)
•	Special Case: Monospace for logs and code
Design Notes:
✅ Clear hierarchy and readability
📐 Layout Grid
Recommended Values:
•	Three-Column Layout: 320px (controls) + 1fr (main) + 350px (observations)
•	Breakpoints: 1200px, 768px (mobile-first)
•	Grid System: CSS Grid with 15px gaps
•	Spacing Units: 15px/20px consistent scales
Design Notes:
✅ Clean, responsive layout for all device sizes
🧊 Border Radius
Recommended Values:
•	Cards/Panels: 15px
•	Components: 12px
•	Buttons: 8px
•	Inputs/Progress Bars: 6px/4px
•	Avatars/Indicators: 50% (circular)
Design Notes:
✅ Cohesive and modern styling
🧱 Elevation
Recommended Values:
•	Level 1: 0 10px 30px rgba(0,0,0,0.1)
•	Level 2: 0 8px 25px rgba(66,153,225,0.3)
•	Level 3: 0 25px 50px rgba(0,0,0,0.15)
•	Interaction: Elevated hover effects
Design Notes:
✅ Intuitive visual depth across layers
Content Design
🖼 Iconography
Recommended Values:
•	Replace emoji-based icons with a consistent SVG icon set (e.g., Feather or Heroicons)
•	Define standard icon sizes and positioning
•	Create icon usage guidelines
⚡ Motion & Interaction
Recommended Values:
•	Hover: translateY(-2px), subtle shadows
•	Loading: Pulse animations
•	Transitions: all 0.3s ease
•	Custom Animations: For specific events
Design Notes:
✅ Smooth UX, could benefit from documented animation principles
♿ Accessibility
Recommended Values:
•	Add ARIA roles and labels
•	Ensure keyboard navigation across all components
•	Improve screen reader support
•	Maintain strong contrast ratios and focus states
Components
🔔 Toast
Recommended Values:
•	Add toast notifications for success, error, and status changes
•	Support positioning (top-right, bottom-left, etc.)
•	Use for ephemeral user feedback
💬 Overlays & Modals
Recommended Values:
•	Maintain existing rotation/failure overlays
•	Add confirmation modals for critical or destructive actions
•	Ensure accessibility (focus trapping, escape key support)
❓ Tooltips & Hints
Recommended Values:
•	Provide tooltips on hover/focus
•	Use consistent styling (dark background, white text)
•	Expand to more UI controls for clarity
